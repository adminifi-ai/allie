Allie backlog 002-005 / executor plan

Branch: deliver/full-backlog

Target outcome

Turn the V0 loop into release evidence infrastructure.

Deliver the remaining backlog without changing Allie's core claim: evidence first, no legal compliance guarantee, model findings non-blocking unless backed by deterministic or human proof.

Chosen Design

Packet-centered progression. Every backlog item extends or projects allie.evidence.v0; no parallel status model.

Fail closed at boundaries. Credentials, artifact redaction, worker failures, missing evidence, and model policy gaps must produce explicit packet status and exit class.

Release projection last. PR/check/release summaries read fixture packet corpora after trust and standards semantics are stable.

Standards

Rust owns orchestration, schema, policy, and projections. Playwright/axe stay behind the worker. No OpenRouter/provider spread.

Proof

Each slice gets a live CLI oracle, focused packet assertions, and a negative fixture that proves the check can fail.

Review

Fresh-context critic sees only the diff, backlog oracle, and live evidence after implementation. Blockers must be fixed and re-reviewed.

Stop If

Stop if a release projection can pass with missing evidence, a model-only finding blocks, secrets enter packet/report output, or CI cannot run the documented gates.
