ALLIE / PLAN 016

autonomy-depth slice - observe act rejudge

BACKLOG 016 - CHILD 3

Agentic review should iterate once when the model needs a bounded observation action.

The current gateway is a single model call over pre-captured media. This slice lets the model request a small safe action, Allie executes it in a fresh page, captures new evidence, and re-judges within the same budget.

CHOSEN DESIGN

Keep browser action execution inside the agentic worker. Permit only non-destructive `press_key` and `wait_ms` review actions; no click, typing, form submit, or app mutation.

PROOF

The fake OpenRouter smoke returns an action on the first call and a verdict on the second. The test fails unless two calls occur and the final assessment includes action-captured media.

CLAIM

Agentic review can observe, act, and re-judge without fabricating verdicts.

FALSIFIER

Only one model call occurs, unsafe actions execute, or final media lacks the action screenshot.

DRIVER

`npm run agentic:smoke`, then `npm run autonomous:smoke` and full `npm run verify`.

STOP IF

The loop requires destructive app actions, model-only release blocking, or provider logic outside the worker.
