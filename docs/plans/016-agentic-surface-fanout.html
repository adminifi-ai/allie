ALLIE / PLAN 016

autonomy-depth slice - surface fanout

BACKLOG 016 - CHILD 4

Agentic review should judge every captured surface, not just the first page.

Allie already records captured state metadata in the evidence packet. This slice passes those states to the agentic gateway, captures per-surface evidence with bounded retries, and aggregates model verdicts so any surface failure wins while inconclusive coverage never becomes a fabricated pass.

CHOSEN DESIGN

Rust owns the deterministic review-surface list from `coverage.state_metadata` and verdict `affected_states`. The worker owns Playwright capture, fake/live provider calls, retry, and final per-obligation aggregation.

QUALITY SYSTEM

Substantive harness change: offline fake-provider smoke, integrated autonomous smoke, full repo gate, and harsh architecture review over the diff.

CLAIM

Agentic review fans out across all captured surfaces and retries transient capture/model failures without weakening no-fabrication policy.

FALSIFIER

The fake OpenRouter run sees only one surface, no retry, unsafe action keys, or a pass when one reviewed surface is missing or inconclusive.

DRIVER

`npm run agentic:smoke`, `npm run autonomous:smoke`, then full `npm run verify`.

STOP IF

Fanout requires provider details in Rust, blocks releases on model-only findings, or needs destructive browser actions.
