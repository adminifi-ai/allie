Allie backlog 001 / executor plan

Branch: deliver/001-v0-evidence-loop

Target outcome

Make Allie's V0 evidence loop executable.

Deliver allie run --manifest examples/login-flow.yml --out .allie/runs/latest so a cold operator gets one checked-in browser flow, axe output, a screenshot, a machine-readable evidence packet, a local HTML report, replay instructions, and deterministic exit behavior.

Chosen Design

Rust core, narrow Node worker. Rust owns CLI parsing, manifest validation, run planning, packet/report writing, hashing, and exit taxonomy. Node owns only Playwright and axe execution behind a request/response JSON boundary.

Local fixture first. A checked-in fixture and no-op auth profile prove evidence semantics before real staging credentials.

Premise sources: SPEC.md, AGENTS.md, docs/architecture.md, docs/evidence-contract.md, backlog.d/001-....

Standards

Evidence packet is the core interface. Rust remains the orchestration owner. Playwright/axe details stay behind the worker adapter. No legal compliance claims.

Acceptance

The V0 command writes evidence.json, report.html, axe JSON, one screenshot, replay instructions, route-state metadata, and exits by deterministic/scripted result.

Verify

Live oracle: V0 command. Structural gates: cargo fmt --check, cargo test --locked, worker smoke. Fresh critic: artifact-only works critique before closeout.

Communicate

Report before edits, after the first live artifact, after gates, and at closeout with exact commands, artifact paths, status, and residual risk.

Stop If

Stop if Playwright cannot install/run locally after two attempts, the evidence contract contradicts the spec, or implementation requires real credentials/model calls to pass.

Current State

The repo has product docs and five active backlog files. The current CLI only prints a product line and next target. No fixture, manifest, worker, packet writer, or smoke command exists.

Change Shape

Add the smallest durable V0 surface: CLI subcommand, manifest schema, evidence schema/types, fixture, worker protocol, report writer, tests, and worker smoke command.

Proof Surface

.allie/runs/latest/ becomes the inspectable receipt: packet, report, screenshot, axe JSON, worker transcript, and replay command.

In Scope

  • Checked-in local flow manifest and fixture app.
  • Playwright plus axe worker with typed JSON request and response.
  • Evidence packet conforming to allie.evidence.v0 shape and formal JSON Schema.
  • Local report with evidence links and replay instructions.
  • Deterministic exit semantics covered by tests.

Non-Goals

  • Real staging auth, secret providers, model review, PR comments, hosted dashboards, remediation branches, video/GIF, and legal compliance claims.
  • Full standards obligation mapping beyond the minimum profile/verdict fields needed by the packet.
  • A general workflow engine or provider abstraction outside the narrow worker and future model gateway boundaries.

Verification System

Claim: one command proves the local deterministic evidence loop works for a checked-in login flow.

Falsifier: missing artifact, invalid packet, broken report link, worker failure, missing screenshot, missing raw axe JSON, unmapped deterministic finding, or wrong exit code.

Driver: cargo run --locked -- run --manifest examples/login-flow.yml --out .allie/runs/latest.

Grader: Rust tests assert manifest validation, packet/report generation, artifact hashes, and exit classes; the worker smoke asserts Playwright plus axe output against the fixture.

Evidence packet: .allie/runs/latest/ plus final command output.

Cadence: after schema/tests, after worker integration, before closeout, and after critic fixes.
