Driver: npm run auth:smoke, Linejam storageState dogfood, then npm run verify.

Grader: gated route URL and marker present, negative control blocks, secret grep clean, and final repo gate green.

Quality System

Standards: no new provider leakage outside the worker auth boundary; no weaker gates; no credential values committed or printed; deterministic fixture-backed proof first.

Proof methods: public CLI smoke, evidence packet inspection, secret grep, real-app receipt, full repo gate.
