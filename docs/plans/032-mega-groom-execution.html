World-class target

A developer (or agent) drops one command into any repo — SPA, HTTPS, authenticated — and gets, in minutes: an autonomously discovered surface map with measured recall; replayed Playwright+axe evidence plus scripted probes settling ~16+ criteria deterministically with the model off; committed, clearly-asterisked agentic verdicts on judgment criteria with calibrated precision (labeled corpus, not vibes); a WCAG 2.2/2.1 ledger with zero silent gaps at both criterion and obligation grain; per-finding SARIF and a schema-versioned JSON contract a fixing agent can act on without re-investigation; an OpenACR projection procurement can file; all from a signed, checksummed release bundle whose doctor never lies. Release decisions ride only the deterministic spine. Nothing is sent to a vendor without an allowlist check, a redaction pass, and an audit event.
