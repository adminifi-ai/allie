Allie / Groom Contract 2026-06-20

Make autonomy long-running and make WCAG visibility exact.

The Vanity dogfood exposed the next product truth: Allie cannot treat a 120-second subprocess as the agentic contract, and it cannot let supporting checks inflate the WCAG denominator. The backlog should now pivot from "all initial epics done" to V1 evidence governance.

Chosen Direction

Agent job runner plus criterion-by-surface matrix. Keep Rust as the evidence engine. Move agent work into resumable jobs with checkpoints, budgets, event logs, and adapters. Report exactly 55 WCAG 2.2 A/AA success criteria separately from Allie supporting checks.

Runtime

Replace wall-clock kill switches with jobs.

Agent sessions should be resumable, cancellable, evented, budgeted, and checkpointed. CI consumes completed packets or bounded deterministic subsets.

Coverage

Matrix every criterion across every surface.

Every WCAG criterion and discovered surface/state gets a status, evidence refs, applicability rationale, confidence, and residual review need.

Landscape

Track competitors by evidence contract.

Deque, Evinced, Level Access, Pa11y, Lighthouse, Accessibility Insights, BrowserStack, Sauce, and Stark should be compared by replayability, governance, and release-decision semantics.
